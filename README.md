# Coaching AI Platform Case Study

受託案件として開発した、**コーチング事業向けAI実行基盤と管理ダッシュボード** の設計・実装事例です。  
対象は、日々の振り返り対話、週次レポート生成、月次レポート生成、承認付き配信、生徒情報の一元管理、過去事例検索です。

このリポジトリでは、コードそのものではなく、**何をどう設計し、どのような制約の中で運用可能な仕組みにしたか** をドキュメント中心に整理しています。

---

## この案件で扱った内容

- 日々の対話ログを起点に、レポート生成、承認、送信までを一つの運用フローとして接続
- 分散していた生徒情報、提出状況、履歴、送信状況を一元的に参照できる管理基盤を構築
- 過去記録を再利用できる検索・分析基盤を導入
- 対外送信を含む運用フローに対して、承認、再送、監査を含む制御を実装
- 継続利用される前提で、運用コスト、保守性、セキュリティを考慮した構成を設計

---

## 成果

- 面談後の週次レポート作成時間を、**週約150分から実質0分** へ削減
- 保護者向け月次レポート作成・送信時間を、**1人あたり約5分から約30秒** へ短縮
- 月次レポート生成の入力を生ログから週次中間成果物へ切り替え、**月次工程の入力 token を約89.8%削減**、**システム全体では約28.2%削減**
- 生徒情報、提出状況、履歴、送信状態を一元管理できるようにし、**複数コーチで運営できる体制** を整備
- 導入前後の月次比較で、**解約率30%低下**
- **新たに4名のコーチを採用可能** になり、**1か月で生徒数30名増加**
